# Beyond-Technical-Artifacts-Multi-Omics-Data
# Supplementary Materials

This repository contains the supplementary materials associated with our research on **biological noise detection and characterization in cancer multi-omics data**.

## Repository Contents

* **Differential Expression Analysis**

  * Raw methylation
  * Reconstructed methylation
  * Raw mRNA
  * Reconstructed mRNA
  * Raw miRNA
  * Reconstructed miRNA

* **Functional Enrichment**

  * Gene Ontology (GO) enrichment results
  * KEGG pathway enrichment results


* **Biological Noise Detection**

  * HDBSCAN clustering results
  * Noise and non-noise sample labels
  * Membership probabilities
  * GLOSH outlier scores
  * Noise Confidence Score (NCS)

* **Figures**

  * Volcano plots
  * Heatmaps
  * PCA visualizations
  * Cluster analysis figures
  * Additional supplementary figures

## Data Availability

The cancer multi-omics datasets analyzed in this study are publicly available from the **National Cancer Institute**. This repository provides only the processed supplementary results generated during our analysis and does not redistribute the original TCGA datasets.
